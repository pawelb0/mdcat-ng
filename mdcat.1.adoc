== Name

mdcat, mdless - render CommonMark Markdown on text terminals

== Synopsis

*mdcat* [_OPTIONS_] [FILE]...

*mdless* [_OPTIONS_] [FILE]

== Description

`mdcat` renders Markdown (CommonMark plus GFM extensions) to a
text terminal. If no `FILE` is given, or if `FILE` is `-`, input
is read from standard input.

Two binaries:

*mdcat*::
    Render to stdout and exit.

*mdless*::
    Open an interactive viewer. See <<mdless>> for keys.

=== Markdown support

CommonMark plus:

* Pipe tables
* Task lists (`- [x]`)
* Strikethrough (`~~text~~`)
* Alert blockquotes (`> [!NOTE]`, `> [!TIP]`, `> [!IMPORTANT]`,
  `> [!WARNING]`, `> [!CAUTION]`)
* Smart punctuation (curly quotes, en/em dash, ellipsis)
* Footnotes (`[^label]`)
* Definition lists (`term\n: definition`)
* Wiki links (`[[Page]]`, `[[Page|label]]`)

Not rendered: math, inline markup inside table cells (cells
truncate with `…`).

=== Terminal detection

Read in order from the environment: `$TERM`, `$TERM_PROGRAM`
(with `$TERM_PROGRAM_VERSION` when required), `$TERMINOLOGY`,
vendor markers (`$WT_SESSION`, `$KONSOLE_VERSION`,
`$TERMINAL_EMULATOR`), multiplexer markers (`$TMUX`, `$STY`).

Non-TTY output drops styling and image escapes.

=== Capability probing

On a TTY with plain ANSI detection, a DA1 query on `/dev/tty`
auto-enables Sixel if the response advertises it. Disable with
*--no-probe-terminal*.

=== Pagination

`mdless` opens the interactive viewer. `mdcat --paginate` pipes
through `$MDCAT_PAGER` / `$PAGER` / `less -r`. Image protocols
are suppressed in both modes.

=== Multiplexers

When `$TMUX` or `$STY` is set, image escapes are wrapped in DCS
passthrough. tmux needs `set -g allow-passthrough on`.

=== Images

Pixel formats decode via the `image` crate. SVG rasterises via
https://github.com/RazrFalcon/resvg[resvg] (static SVG 1.1).
Terminology renders SVG natively. Images over 100 MiB are
skipped.

=== Remote resources

HTTP(S) images are not fetched by default. Pass
*--remote-images* to opt in.

== Options

Common to both binaries unless noted.

*-c, --no-colour, --no-color*::
    Disable all styling. Implied when stdout is not a TTY.

*--ansi*::
    Use ANSI formatting regardless of detected terminal.
    Conflicts with *--no-colour*.

*--columns=_COLUMNS_*::
    Render at the given column width.

*--remote-images*::
    Fetch HTTP/HTTPS images for inline display. Off by default;
    remote images render as OSC 8 hyperlinks otherwise.

*-l, --local*::
    Deprecated. No-op alias kept for 2.x command lines.

*--fail*::
    Exit on the first file that fails to load.

*--detect-terminal*::
    Print the detected terminal, multiplexer, and probed Sixel
    support, then exit.

*--no-probe-terminal*::
    Skip the DA1 probe.

*--probe-timeout-ms=_MS_*::
    DA1 reply timeout in milliseconds (default: 50).

*--wrap-code*::
    Wrap code-block lines instead of overflowing.

*--completions=_SHELL_*::
    Generate completions for _SHELL_ (`bash`, `zsh`, `fish`,
    `powershell`, `elvish`) to stdout and exit.

*-h, --help*::
    Show help and exit.

*-V, --version*::
    Print the version and exit.

=== Options for `mdcat`

*-p, --paginate*::
    Pipe output through `$MDCAT_PAGER` / `$PAGER` / `less -r`.

*-P, --no-pager*::
    Do not paginate (default).

=== Options for `mdless`

*-P, --no-pager*::
    Skip the interactive view and print to stdout.

*--external-pager*::
    Shell out to `$MDCAT_PAGER` / `$PAGER` / `less -r`.

*--search=_PATTERN_*::
    Commit a search before the event loop starts. Combines with
    *--regex* and *--case-sensitive*.

*--regex*::
    Interpret the search pattern as a regex.

*--case-sensitive*::
    Force case-sensitive matching (default is smart-case).

*-n, --line-numbers*::
    Show a rendered-line number gutter. Toggle live with `#`.

== mdless keybindings

| Key | Action
| `j`, `k`, Down, Up | Scroll one line
| Space, `b`, PageDown, PageUp | Page forward / back
| Ctrl-F, Ctrl-B | Page forward / back
| Ctrl-D, Ctrl-U | Half page forward / back
| `g`, `gg` | Jump to top
| `G` | Jump to bottom
| _N_`G` | Jump to line _N_
| `0`, Home | First line
| End | Last line
| `/` _PATTERN_ Enter | Forward search
| `?` _PATTERN_ Enter | Backward search
| `n`, `N` | Next / previous match
| `Esc` | Clear highlights
| `]]`, `[[` | Next / previous heading
| `T` | Toggle TOC modal
| Enter (in TOC) | Jump and close
| `m`_{a-z}_ | Save bookmark
| `'`_{a-z}_ | Jump to bookmark
| `#` | Toggle line numbers
| Ctrl-L | Redraw
| `q`, Ctrl-C | Quit

== Exit status

`mdcat` exits `0` on success, `1` on rendering errors (or on the
first failure when *--fail* is passed), `128` if the external
pager fails to start.

`mdless` exits `0` on clean quit, `1` on failure to open or
render input.

== Environment

*TERM*, *TERM_PROGRAM*, *TERM_PROGRAM_VERSION*, *TERMINOLOGY*::
    Terminal detection.

*WT_SESSION*::
    Windows Terminal marker.

*KONSOLE_VERSION*::
    KDE Konsole marker.

*TERMINAL_EMULATOR*::
    Contour marker (when set to `contour`).

*TMUX*, *STY*::
    Trigger DCS passthrough wrapping.

*COLUMNS*, *ROWS*::
    Fallback size when `ioctl(TIOCGWINSZ)` fails.

*MDCAT_PAGER*, *PAGER*::
    Command for `mdcat --paginate` and `mdless --external-pager`.
    Default: `less -r`. Empty `$MDCAT_PAGER` disables pagination.

*http_proxy*, *https_proxy*, *HTTPS_PROXY*, *all_proxy*, *ALL_PROXY*, *no_proxy*, *NO_PROXY*::
    libcurl proxy settings. See *curl(1)*.

*MDCAT_LOG*::
    Tracing filter. `MDCAT_LOG=trace` enables full tracing;
    `MDCAT_LOG=mdcat::render=trace` narrows the scope.

== Examples

*Render a file*::
    mdcat README.md

*Read from stdin*::
    curl -sL https://example.com/doc.md | mdcat -

*Open at first match*::
    mdless --search "## Installation" README.md

*Detect the terminal*::
    mdcat --detect-terminal

*Fixed width, no styling*::
    mdcat --columns 72 --no-colour README.md | pbcopy
